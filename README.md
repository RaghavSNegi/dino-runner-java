# 🦖 Dino Runner (Java)

A simple endless runner game inspired by Chrome's offline dinosaur game, built using Java Swing.

## 🎮 Features

- Jump mechanics with smooth physics
- Dynamic obstacle generation (cactus)
- Increasing difficulty over time
- Collision detection system
- Score tracking
- Sound effects (jump & game over)
- Visual elements: sun, clouds, ground

## 🕹 Controls

- **SPACE** → Jump / Restart

## 🛠 Tech Stack

- Java
- Swing (GUI)
- AWT (graphics)
- Java Sound API

## 🚀 How to Run

1. Clone the repository
2. Open in VS Code / IntelliJ
3. Run `Game.java`
